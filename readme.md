# Greens Drawer

A simple web app to draw on an IGN or ESRI map, export/import GeoJSON, and visualize your data.

- Draw polygons and markers on the map
- Edit or delete your drawings
- Copy/paste GeoJSON in the right panel
- Switch between IGN and ESRI basemaps
